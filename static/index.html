👋 Bienvenido a QSL Manager

Primero, ¿cuál es tu indicativo? Así la app sabe reconocer que las tarjetas están dirigidas a vos (y no confundirlo con el remitente). También aparece arriba en la pantalla, útil si compartís capturas.

Esta app funciona con un motor de reconocimiento local (Tesseract) -- no necesitas configurar más para empezar a importar tus QSLs.

Importar desde eQSL.cc

Baja las tarjetas de tu bandeja de entrada de eQSL.cc directamente. Tus datos vienen ya confirmados (indicativo, fecha, banda, modo), así que estas tarjetas quedan marcadas como verificadas sin necesidad de reconocimiento. eQSL no permite descargas rápidas -- esto baja una tarjeta cada ~2 segundos, así que un inbox grande puede tardar varios minutos; puedes cerrar esta ventana y seguir usando la app mientras tanto, el progreso sigue en segundo plano.

Se guardan solo en tu PC (data/config.json), igual que la API key de Gemini. eQSL no ofrece nada más seguro que usuario/contraseña para esto.
